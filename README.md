# Home Assistant Carrier Board

**Custom carrier board for Raspberry Pi Compute Module 4 – Optimized for Home Assistant**

## Overview

This project is a custom-designed carrier board for the **Raspberry Pi Compute Module 4 (CM4)**, specifically built to run **Home Assistant** efficiently and reliably.

Similar to the official CM4 IO Board or the well-known “Tofu” carrier, this board provides a compact and practical way to expose the CM4’s interfaces while adding features tailored for home automation use.

## Key Features

- Raspberry Pi Compute Module 4 socket
- Multiple USB ports (including USB-A)
- Gigabit Ethernet (or support via CM4)
- HDMI output
- M.2 PCIe slot for NVMe SSD (256 GB shown in photo)
- CSI and DSI connectors for camera and display
- Micro-SD card slot
- Power management with wide input voltage support
- Status LEDs and convenient connectors
- Compact layout ideal for 24/7 operation in a home server enclosure

## Project Goal

The main purpose of this board is to create a clean, reliable, and compact **Home Assistant server**:

- Fast boot and storage thanks to the NVMe SSD
- Easy integration of cameras, Zigbee/Z-Wave sticks, and other smart home peripherals via USB
- Stable 24/7 operation with good power delivery
- Simplified installation in a small case or wall-mounted enclosure

It serves as a solid foundation for a powerful, low-power smart home hub running Home Assistant OS.
